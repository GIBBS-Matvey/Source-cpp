### Идея решения:
Словарь отсортирован в лексикографическом порядке, а это значит, что для каждого запроса достаточно находить первое слово, начинающегося с заданного префикса (так называемый `lower_bound`), и отсчитывать от него слово с заданным порядковым номером. Ассимптотика `lower_bound` это $\log N$.
